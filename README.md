# RD Station Challenge

## Pré-requisitos

- Node.js (versão 14 ou superior)
- npm (versão 6 ou superior) ou yarn

## Comandos

- Instale as dependências e execute a aplicação:

```bash
npm install
```

```bash
npm run dev
```

Agora acesse `localhost:5173`

- Ou, se preferir, faça o build do código:

```bash
npm run build
```
